Three docs relocated rather than deleted to follow the existing project convention (`doc/archive/` already contains obsolete docs like `FINAL_REPORT.md`, `NEXT_STEPS.md`, etc.). No references to these files exist outside themselves (README and root docs are unaffected).

n8n-workflow was initially bumped to 2.x but rolled back: its CJS build references unbundled `.ts` files in `src/`, which Jest cannot resolve.
