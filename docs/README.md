## 📋 Quick Reference

### Installation
```bash
npm install n8n-prestashop8-node
```

### Basic Configuration
```javascript
{
  "baseUrl": "https://your-store.com/api",
  "apiKey": "YOUR_API_KEY"
}
```

### First Test
```javascript
{
  "resource": "products",
  "operation": "list",
  "advancedOptions": { "limit": "5" }
}
```
